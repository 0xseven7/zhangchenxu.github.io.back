内建类型实例化

JS中提供少量的内建类型供大家方便使用：

  • Object 通用对象类型
  • Date 时间和日期类型
  • Error 运行错误类型
  • Function 函数
  • RegExp 正则表达式

每个内建类型实例化都有两种方法：new操作符创建和字面量创建。

Object类型

创建Object实例的方式有两种：

  1. 使用new操作符后面跟Object构造函数；

  2. 使用对象字面量

两种方法效果相同，但是字面量定义的方法不在调用Object构造函数了。

栈方法和队列方法

栈：后进先出(Last-In-First-Out)的数据结构，最后进去的最先出来。
队列： 先进先出 (First-In-First-Out)的数据结构，最先进去的最先出来。

  1. 栈方法；
    push()接受任意数量的参数，并添加到数组末尾，返回添加之后的数组长度；
    pop()从数组末尾移除一项，兵返回移除的那一项

以上代码中的数组就可以看成是栈；

  1. 队列方法 ：
    shift()从数组头部移除一项，并返回这一项。
    unshift()从数组头部添加任意项，兵返回数组长度。
    shift和push配合使用可以队列一样使用数组

可见sort()方法是通过比较字符串来进行排序的。但sort()可以接受一个比较函数，

操作方法

concat()可以基于当前数组创建新的数组，而且不影响原来的数组

slice能基于当前数组创建一个或者多个新的数组。可以接受一个或两个操作数，如下

splice是最强大的操作方法，可以新增，删除，更改数组项，这个操作会返回删除的项，并更改原来数组。
splice接受多个参数，splice(a, b, "", "")

  • 删除： splice(0,2)指定两个参数，要删除的第一项和要删除的项数
  • 插入 splice(1, 0 , str1, str2) 把第二个参数指定为0，表示要删除0项，也就是插入。
  • 替换 splice(1, 3, str1, str2, str3 )删除第二项之后的三项，并插入str1, str2, str3；
