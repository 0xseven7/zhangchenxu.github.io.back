高效使用windows

发表于 2016-11-08   |  

####如何优雅的使用windows(win10)
@(网页学习)[win10]

写在前面

以下为本人学习工作中总结出的习惯，仅提供参考

##一、高效使用

快捷键

通用部分

  • ctrl+c、ctrl+v、ctrl+z、ctrl+x复制粘贴撤销剪切；
  • shift+delete永久删除，别乱用啊；
  • ctrl+shift+Esc调出任务管理器；
  • win+r打开运行；
  • win+e打开计算机；
  • win+d隐藏所有窗口，快速显示桌面；
  • win+↑最大化当前窗口；
  • win+↓窗口化当前窗口；
  • win+←窗口最大化左半屏幕；
  • win+→串口最大化右伴屏幕；
  • win+space切换输入法；
  • win+i打开设置；
  • win+L锁定计算机；
    未完。

    win10部分

    win10最有特色的就是可以创建多个桌面，也就是多个工作区，每个工作区互相不干扰。
  • win+ctrl+D创建新的桌面/工作区；
  • win+ctrl+←→可以快速切换工作区；
  • win+Tab在本工作区中预览窗口；
  • alt+Tab在本工作区中快速切换窗口；
  • win+c唤出小娜；
    未完。

二、软件推荐

有一套好的应用软件。除了要安装合适的软件之外，还要花费一些精力进到软件里设置一下使用偏好，比如界面结构。虽然很多国产软件颇遭诟病，但毕竟还是有良心之作的，比如网易云音乐。以下是我自己电脑里的基础软件清单（基本都是些大众软件）

原则：

以下是我总结出来的软件选择优先级的原则：
重要一点： 安装软件的时候时刻警惕，避免被捆绑安装。

  • 首选B/S软件，如chrome的扩展程序。轻量级，可跨平台。
  • 如果存在网页版，在不影响使用的情况下使用网页版，方便快捷，跨平台。
  • 如果网页版，在不影响使用的情况下使用网页版，方便快捷，跨平台。
  • 优先使用绿色免安装版
  • 如果必须使用国产软件时，选择纯净版。如QQ纯净版，没广告又轻量，必须功能都有。
  • 如果必须使用国产软件且没有纯净版，那就只好安装使用的时候保持警惕，不要被全家桶伺候（尤其是某些流氓软件）。
  • 获取软件的时候尽量去官网下载。

软件推荐

windows store 和自带的那些 uwp 也都是及其优雅的东西。当然指的是win10

  1. eagleget
    替代迅雷，现在迅雷越做越差劲，最新版本广告甚多，而且不知道迅雷怎么给自己定位的，有点想做浏览器的感觉。
    优点：软件界面清洁优雅，无广告，定位i准确，就是下载工具，有视频探诱功能。
    缺点： 不能通过BT下载。这是可以使用QQ旋风。
  2. QQ旋风
    特点：轻量小型，支持BT，腾讯的良心软件。
  3. chrome
    搭了梯子之后（如何翻墙），可使用各种插件，尤其是现在BS软件越来越多，可满足很多跨平台使用需求。
    介绍几个插件吧：
  • send to kindle: 各种网页都可以转存到kindle里面拉；
    剪藏到印象笔记：各种网页都能转存到印象笔记里面拉；
  • 马克飞象：使印象笔记支持markdown语法，写起笔记来得心应手，付费的可以自动保存到印象笔记。
  • Gmail Checker：gmail邮件提醒插件；
  • Gliffy Diagranms： 画流程图。
  1. firefox
    firefox在没梯子的时候也能使用各种插件，尤其是神器firebug，但是firefox相较于chrome有点臃肿，并不是太常用。
  2. QQ输入法纯净版
    特点：安装包小、速度快，并且无弹窗、无广告、轻负担，去除了多项冗余操作。
  3. 微软自带的输入法
    最新的微软拼音比之前的那是很多了，可以尝试一下。
  4. Listary
    window自带的文件搜索那叫是一个鸡肋啊，费时费力还找不到。有了这个神器之后，搜索文件那是分分钟全盘搜索。
  5. Xmind
    制作各种漂亮的思维导图，理清思路，功能超级全面。
  6. picpick
    图片处理软件，功能全面到令人发指，从截图，取色，到简单的处理图片，应有尽有。
  7. 印象笔记
    同类产品有为知笔记，有道云笔记和微软的onenote。我个人还是喜欢印象笔记，个人偏好吧。某宝上10几块钱可以买一年的会员，配合马克飞象可支持Markdown格式写作。
  8. 网易云音乐
    和网易新闻截然不同的网易的产品，有点自己体会吧。
  9. Potplayer
    之前快播没被查封之前，我一直使用快播作为播放软件，就是因为快播播放功能强大，快进快退不会出现卡顿，也不会出现广告什么的。但没了快播之后，Potplay和KMplay成了首选。
  10. QQ 精简版
    必要功能：邮箱，空间等全都有，去除了很多没必要的功能，最重要的是没有弹窗广告。
  11. 坚果云
    国内有很多提供云服务的厂家，比如百度云，360晕，微云。这些厂家标榜的容量都很惊人，动不动就几百上千G的。但是这也只是标榜，其实很多文件在内部只能算是“超链接“。由于上传和下载速度慢得惨绝人寰，再加上“免费的是最贵的”原则，我选择了坚果云。
    特点：限流量，但不限速，传输稳定快捷。没有客户端，客户端就是文件夹。
  12. github
    全球最大同性交友网站（手动眼斜）
  13. f.lux
    屏幕调色软件，它的过人之处是会随着一天24h不断改变屏幕色温，达到保护眼睛的效果。
  14. Ad-safe
    掉渣天的广告过滤软件，可以屏蔽90%的广告，尤其是可以跳过视频中的广告，从此干干净净再也没有烦人的广告啦。
  15. Bandizip解压软件
  16. Foxit Reader
    免费的福新PDF阅读器，比Adobe reader更小更强悍。

    三、优化

    需要的话，使用固态硬盘。
    不用杀软
    其实最重要的是, 自己的文件自己要时常整理。

    windows只是平台，其实真正使用的是上面跑的程序。
