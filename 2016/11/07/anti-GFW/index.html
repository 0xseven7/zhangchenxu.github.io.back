使用xx-net翻墙

张晨旭
2016-11-07

简单的翻墙
使用chrome + xx-net搭建梯子。

下载chrome

chrome

下载XX-net

XX-net

主要特性

集成GoAgent、GoGoTest等自动扫描IP，省事省力；
Web界面，人性化交互，傻瓜易用；
内置了公共 appid, 方便新手；
方便易用的细节、细节提示；
平台支持情况：
Windows 7/8/10 （xp 需要 tcpip.sys 补丁, 比如用 tcp-z）；
Linux （Ubuntu不显示系统托盘）；
Mac OS X(10.7; 10.8; 10.9; 10.10)

配置方法

Windows下, 双击 start.lnk/start.vbs，启动弹出浏览器： 访问 http://localhost:8085/ ，点击托盘图标可弹出Web管理界面, 右键可显示常用功能菜单。
Win7/8/10：提示请求管理员权限, 安装CA证书。请点击同意。推荐用Chrome浏览器, 安装SwichySharp, 可在swichysharp目录下找到插件和配置文件，Firefox 需手动导入证书 data/gae_proxy/CA.crt 启动后生成。
虽然系统内置了公共appid, 还是建议部署自己的appid，公共appid限制看视频。

使用方法

双击快捷方式即可翻墙。
