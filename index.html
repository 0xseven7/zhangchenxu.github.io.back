NodeJS学习

构建简单的HTTP服务器：

Node中的http模块提供了HTTP服务器和客户端的接口，创建HTTP服务器需要调用http.createServer()函数，他有一个参数，十个回调函数，服务器每次接收到HTTP请求的时候都会调用这个函数，这个请求回调函数有两个参数，req（请求）和res（响应）；

服务器接收到一条HTTP请求的时候，就会用新的req和res对象触发回调函数，在触发之前，Node会先解析请求的http头，并将他们作为req对象的一部分提供给回调函数，但在触发回调函数之前，Node不会解析请求体的内容。
Node不会自动往客户端里卖年写任何相应，也就是说它不会自动结束响应。在调用完请求回调函数之后，就要由你负责用res.end()方法来结束响应。

  • 获取软件的时候尽量去官网下载。

TCP/IP协议

计算机网络设备之间的相互通信，肯定需要一种规则，因为这涉及到很多不同的硬件，不同的操作系统，这种规则就是协议(protocol)；
TCP/IP 协议就是与互联网相关联的协议的集合，也有人认为TCP/IP协议就是TCP/IP和IP这两种协议。

TCP/IP分层

TCP/IP协议按层可以分为四层：

  • 应用层
  • 传输层
  • 网络层
  • 数据链路层
应用层 （Application Layer）

应用层位于TCP/IP协议族的最上层，最靠近用户，决定了向用户提供服务时通信的活动。
应用层预存了各类通用的应用服务

  • FTP （File Transfer Protocol，文本传输协议）
  • DNS （Domain Name Sytem， 域名解析系统）
  • HTTP
传输层 （Transport Layer）

传输层在应用层上层，提供处于处于网络链接中的两台计算机的数据传输，有两个性质不同的协议 ： TCP （transmission Controll） 传输控制协议 和 UDP（User Data Protocol ）用户数据协议

网络层 （Internet Layer）

网络层用来处理在网络上流动的数据包。数据包时网络传输的最小数据单位。该层规定了通过怎样的路径到达对方的计算机，并把数据包传送给对方。
