webstorm 技巧整理

常用快捷键

界面操作

  • ctrl + alt + s 打开设置
  • shift + F6 文件重命名
  • ctrl + tab 快速切换选项卡
  • alt + ← or → 左右切换选项卡
  • ctrl + F12 显示文档的结构
  • ctrl + E 最近打开的文件
  • alt +ins 添加新文件

    coding

  • ctrl + d 复制到向一行
  • ctrl + enter 无论光标在何处, 均向上开辟新的一行
  • shift + enter 同上, 向下开辟一行
  • ctrl + x 剪切
  • ctrl + v 粘贴
  • ctrl + c 复制
  • ctrl + y 删除
  • ctrl + del 删除到单词结束
  • ctrl + back 删除到单词开始
  • ctrl + w 选中单词
  • f2/ shift + f2错误定位
  • ctrl + alt + l 格式化代码

一些设置方法

import/export 可以导入或者输出.jar的设置
filetype可以添加文件的类型, 并规定用什么格式打开.
file and code template 可以添加模板, 用于新建某种文件的时候使用该模版.
live template 自定义模板, 用于快速书写代码

破解

支持正版, 希望有能力的开发者购买正版.
