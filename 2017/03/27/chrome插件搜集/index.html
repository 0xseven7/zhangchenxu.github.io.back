chrome插件搜集

张晨旭
2017-03-27
chrome extentions

由于chrome插件不能像收藏夹那样和账号同步, 在每次重新安装之后总是要寻找好久之前使用的插件. 现在记录一下.

日常使用

  • adblock plus
    广告拦截神器
  • simple undo close
    之前使用其他浏览器的时候感觉恢复按钮特别好用, 虽然chrome有ctrl+shift+T来恢复上一个打开的页面, 但不够直观. 这个插件可以完美解决这个问题.
  • papaly
    牛x的书签管理器, 支持同步
  • onekeyplugin
    一键管理所有插件, 特色是一斤禁用所有插件再解禁的时候, 可以保持之前插件的禁用/使用状态, 如果是禁用状态就保持禁用.
  • crxMouse Chrome Gestures
    鼠标手势
  • one tab
    有时候会开很多的标签,而chrome又特别吃内存. onetab可以讲标签全部整合, 供以后再打开, 节约内存.
  • scroll style
    chrome的滚动条有点难看, 这个插件可以让它像手机的滚动条一样漂亮.
  • bilibili助手
    b站一直是我喜欢的网站,很喜欢看着弹幕和大家一起看视频的感觉. 但是现在弹幕质量越来越低, 此插件可以很好的帮我清理垃圾弹幕, 改善视频体验.
  • tempermonkey

    工作学习

  • 马克飞象
    印象笔记很好用, markdown也很牛x, 但是无奈的是印象笔记比支持markdown语法. 马克飞象是chrome的扩展应用, 一款markdown的编辑器, 可以同步到印象笔记.
  • postman
    发送请求, 快速测试调试
  • 印象笔记剪藏
    使用印象笔记扩展程序一键保存精彩网页内容到印象笔记帐户. 也可通过这个插件更好的阅读网页
  • mail checker for google
    gmail的插件, 无需打开 Gmail 或 Inbox，即可收到桌面邮件通知，方便地查看、收听或删除邮件，并且支持多账户。
  • diigo
    互联网上的知识大多是碎片化的, 很多时候需要化零为整, diigo可以在阅读网页的时候进行批注和整理.
  • web developer
    一些常用的web开发工具
  • 前端助手
    FE助手：包括字符串编解码、代码压缩、美化、JSON格式化、正则表达式、时间转换工具、二维码生成器、编码规范检测、页面性能检测、页面取色、Ajax接口调试
  • vue
    调试vue的插件, 可以清晰的看到父子之间的data, prop, event等, 很好用!
  • 作图
    • gliffy
    • draw.io
      两个扩展程序的都都非常好用, 都支持同步, draw.io可以同步到github的项目中.
