运行中(win + r) key 与 app 对照表

张晨旭
2017-03-30
win10

平时工作学习中, 多使用win + r进入运行, 键入key, windows就可以打开key对应的app, 文件或者文件夹(需配置). 这些操作可以帮助我们更快的打开我们想要打开的东西, 节约时间, 解脱鼠标.
